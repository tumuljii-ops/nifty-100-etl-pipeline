# NIFTY 100 Financial Analytics Platform

## Overview

A Python-based financial analytics platform built during a Data Analytics Internship.

The project performs ETL, financial ratio computation, stock screening, peer comparison, valuation analysis, and interactive visualization using Streamlit.

---

## Technology Stack

- Python
- Pandas
- SQLite
- Streamlit
- Plotly
- OpenPyXL
- Matplotlib

---

## Features

- ETL pipeline for financial datasets
- Financial ratio engine
- CAGR calculations
- Cash Flow KPI engine
- Stock Screener
- Peer Comparison Engine
- Valuation Engine
- Interactive Streamlit Dashboard
- Excel & CSV report generation

---

## Dashboard

streamlit run src/dashboard/app.py
